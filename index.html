About

Visual Web Comic Dubber is a Chrome extension that enhances your web comic reading experience by automatically detecting speech bubbles, extracting text using Google Gemini 2.0 Flash API for OCR, and reading it aloud using the Web Speech API.

With customizable voices, speeds, and character-specific settings, you can enjoy web comics in a whole new way, making them more accessible and engaging.

Installation Instructions

1 Clone or download the repository

Get the code from GitHub by cloning the repository or downloading as a ZIP file.

2 Set up the backend server

Navigate to the backend directory, install dependencies with npm install, create a .env file with your Gemini API key, and start the server with npm start.

3 Load the extension in Chrome

Open Chrome and go to chrome://extensions/, enable "Developer mode", click "Load unpacked", and select the extension directory.

4 Configure settings (optional)

Click on the extension icon and go to "Advanced Settings" to customize voices, speeds, and character assignments.

How to Use

1 Navigate to a web comic page

Open your favorite web comic in Chrome.

2 Click the extension icon

Click on the Visual Web Comic Dubber icon in your browser toolbar to open the popup.

3 Use the controls

Use the play, pause, and stop buttons to control the text-to-speech playback.

4 Adjust settings

Click the settings icon to customize voice, speed, and pitch, or access advanced settings for more options.
